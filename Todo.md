# 基础数据

> 整个钱包的数据来源内容
> 核心为：
>   主链 => 决定接口请求服务
>   钱包 => 后续操作的基础依赖

- 主链数组 chainList
> 主链可以自定义添加，所以需要通过本地化配置存储和读取

```
[
	{
		name: 'Cosmos', // 不能重复，作为唯一值
		lcd: 'xxxx',
		logo: 'xxxx', // 没有则取默认值
		version: '',
		wallets: []
	}
]

```

- 钱包数组  walletList

> 每个主链下都有一个钱包数组

```
[
	{
		name: '', // 可重复
		chain: 'Cosmos',
		address: '',
		addTime: '' // 添加时间，可作为本地的唯一识别
	}
]
```
